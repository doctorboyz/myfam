# คู่มือเริ่มต้นใช้งาน MyFam

เอกสารนี้อธิบายขั้นตอนการเริ่มใช้งานแอป MyFam สำหรับผู้ใช้ใหม่ ตั้งแต่เข้าใช้งานครั้งแรกจนถึงการใช้งานฟีเจอร์หลักทั้งหมด

---

## 1. เข้าใช้งานครั้งแรก

### ล็อกอินด้วย LINE
1. เปิดแอป MyFam ผ่าน LINE (LIFF) หรือเปิดลิงก์ที่ได้รับจากผู้ปกครอง
2. กด **"เข้าสู่ระบบด้วย LINE"**
3. อนุญาตให้แอปเข้าถึงข้อมูลโปรไฟล์ LINE ของคุณ (ชื่อ รูปโปรไฟล์)
4. ระบบจะสร้างบัญชีผู้ใช้อัตโนมัติหากยังไม่เคยใช้งาน

> **หมายเหตุ:** ผู้ใช้ใหม่ที่ยังไม่ได้เชื่อมบัญชีจะถูกนำไปยังหน้า `/link` เพื่อขอลิงก์เชื่อมต่อจากผู้ปกครอง

---

## 2. สร้างครอบครัว (สำหรับ Parent)

เมื่อล็อกอินครั้งแรก ผู้ใช้ที่มีบทบาท **Parent** จะต้องสร้างครอบครัว:

1. กด **"สร้างครอบครัว"**
2. ตั้งชื่อครอบครัว เช่น "ครอบครัวสมชาย"
3. ระบบจะสร้างครอบครัวและตั้งคุณเป็น **ผู้ปกครอง (Parent)** โดยอัตโนมัติ

> ผู้ปกครองสามารถจัดการสมาชิก บัญชี และงบประมาณทั้งหมดในครอบครัวได้

---

## 3. เพิ่มสมาชิกในครอบครัว

### สำหรับ Parent
1. ไปที่ **การตั้งค่า > สมาชิกครอบครัว**
2. กด **"เพิ่มสมาชิก"**
3. กรอกชื่อสมาชิกและเลือกบทบาท:
   - **Parent** — ผู้ปกครอง (จัดการทุกอย่างได้)
   - **Child** — บุตร (ดูข้อมูลและบันทึกรายการได้)
4. กด **"บันทึก"**

### ส่งคำเชิญเชื่อมบัญชี LINE
1. ในหน้าสมาชิกครอบครัว สมาชิกที่ยังไม่ได้เชื่อม LINE จะมีปุ่ม **"ส่งคำเชิญ"**
2. กดปุ่ม → ระบบจะสร้าง invite code (8 ตัวอักษร)
3. ส่งลิงก์ให้สมาชิกผ่าน LINE:
   - กด **"คัดลอกลิงก์"** แล้วส่งในแชท
   - หรือกด **"ส่งผ่าน LINE"** (หากอยู่ใน LIFF)
4. สมาชิกเปิดลิงก์และกดยืนยัน → เชื่อมบัญชีสำเร็จ

> Invite code มีอายุ 24 ชั่วโมง และใช้ได้ครั้งเดียวเท่านั้น

---

## 4. สร้างบัญชี (Account)

1. ไปที่แท็บ **"บัญชี"**
2. กดปุ่ม **"+"** (มุมขวาล่าง)
3. กรอกข้อมูลบัญชี:
   - **ชื่อบัญชี** — เช่น "กระเป๋าสตางค์", "ธนาคารกสิกร"
   - **ประเภท** — เงินสด (Cash) / ธนาคาร (Bank) / บัตรเครดิต (Credit)
   - **ยอดเริ่มต้น** — จำนวนเงินในบัญชีตอนนี้
   - **เลือกไอคอน** — เพื่อจดจำง่าย
4. กด **"บันทึก"**

> สร้างได้หลายบัญชี เช่น เงินสด, บัญชีธนาคาร, บัตรเครดิต

---

## 5. บันทึกรายการธุรกรรม

1. ไปที่แท็บ **"แดชบอร์ด"** หรือเปิดบัญชีที่ต้องการ
2. กดปุ่ม **"+"** (มุมขวาล่าง)
3. เลือกประเภทรายการ:
   - **รายรับ (Income)** — เงินเข้า เช่น เงินเดือน, ค่าขนม
   - **รายจ่าย (Expense)** — เงินออก เช่น ซื้อของ, ค่าอาหาร
   - **โอนเงิน (Transfer)** — ย้ายเงินระหว่างบัญชี
4. กรอกรายละเอียด:
   - จำนวนเงิน
   - หมวดหมู่ (เช่น อาหาร, การเดินทาง, ของใช้)
   - เลือกบัญชี
   - วันที่และเวลา
   - หมายเหตุ (ถ้ามี)
5. กด **"บันทึก"**

> หากบันทึกผิด สามารถกดที่รายการแล้วเลือก **"ลบ"** หรือแก้ไขได้

---

## 6. สร้างงบประมาณ (Budget)

1. ไปที่แท็บ **"งบประมาณ"**
2. กดปุ่ม **"+ สร้างงบประมาณ"**
3. เลือก **วัตถุประสงค์**:
   - **ใช้จ่าย (Spending)** — กำหนดวงเงินใช้จ่าย เช่น ค่าอาหารต่อเดือน
   - **ออมเงิน (Savings)** — ตั้งเป้าออม เช่น ออมเงินไปเที่ยว
   - **รางวัล (Reward)** — ให้รางวัลเงินเมื่อบุตรทำตามเป้า
4. กรอกข้อมูล:
   - ชื่องบประมาณ
   - เป้าหมายจำนวนเงิน
   - บัญชีที่เกี่ยวข้อง (สำหรับ savings/reward)
   - สมาชิกที่เกี่ยวข้อง (สำหรับ reward)
   - วันที่เริ่มต้น-สิ้นสุด
5. กด **"บันทึก"**

### ติดตามความคืบหน้า
- แต่ละงบประมาณจะแสดง **แถบความคืบหน้า** (progress bar)
- กดเข้าไปดูรายละเอียด → เห็นรายการธุรกรรมที่เกี่ยวข้องและสถานะ
- งบประมาณที่ครบเป้าจะแสดงแถบสีเขียวและข้อความ "ครบเป้าหมายแล้ว!"

---

## 7. ดูภาพรวมและรายงาน

### แดชบอร์ด
- แสดง **ยอดรวมทั้งหมด** ของครอบครัว
- แสดง **รายการล่าสุด** (รายรับ/รายจ่าย/โอนเงิน)
- แสดง **บัญชีทั้งหมด** พร้อมยอดเงิน

### หน้าบัญชี
- ดูรายการธุรกรรมทั้งหมดของแต่ละบัญชี
- กดที่รายการเพื่อดูรายละเอียด
- แท็บ **"กระทบยอด"** — ดูประวัติการปรับยอดบัญชี

---

## 8. กระทบยอดบัญชี (Reconcile)

ใช้เมื่อยอดในแอปไม่ตรงกับยอดจริง (เช่น ลืมบันทึกรายการ หรือมีดอกเบี้ยเข้า):

1. เปิดบัญชีที่ต้องการกระทบยอด
2. กดไอคอน **refresh** ที่มุมขวาของการ์ดบัญชี
3. กรอก **ยอดเงินจริง** ที่บัญชีมีอยู่ตอนนี้
4. ระบบจะคำนวณส่วนต่างและบันทึกประวัติไว้
5. กด **"ยืนยัน"**

> ดูประวัติการกระทบยอดได้ที่แท็บ **"กระทบยอด"** ในหน้าบัญชี

---

## 9. การแจ้งเตือนผ่าน LINE

### เพิ่ม LINE Bot เป็นเพื่อน
1. หลังจากเชื่อมบัญชีสำเร็จ หน้าจอจะแสดงปุ่ม **"เพิ่มเพื่อน LINE"**
2. กดปุ่ม → เปิดหน้าเพิ่มเพื่อนใน LINE
3. เพิ่มบอตเป็นเพื่อนเพื่อรับการแจ้งเตือน

### การแจ้งเตือนที่จะได้รับ
- รายการธุรกรรมใหม่ในครอบครัว
- งบประมาณใกล้ครบเป้าหรือครบเป้า
- แจ้งเตือนจากผู้ปกครอง

> หากไม่เพิ่มเพื่อน แอปใช้งานได้ปกติ แต่จะไม่ได้รับแจ้งเตือนทาง LINE

---

## 10. การตั้งค่าและจัดการครอบครัว

### ไปที่แท็บ "การตั้งค่า"
- **สมาชิกครอบครัว** — เพิ่ม/แก้ไข/ลบสมาชิก
- **แท็ก** — จัดการหมวดหมู่รายการ
- **โปรไฟล์** — แก้ไขชื่อและข้อมูลส่วนตัว

### สำหรับ Parent เท่านั้น
- สร้าง/ยกเลิก invite code
- ลบบัญชีหรือรายการธุรกรรม
- จัดการงบประมาณทั้งหมด

---

## สรุปลำดับขั้นตอนเริ่มต้น

```
1. ล็อกอินด้วย LINE
   ↓
2. สร้างครอบครัว (Parent)
   ↓
3. เพิ่มสมาชิก → ส่ง invite code
   ↓
4. สร้างบัญชี (Cash / Bank / Credit)
   ↓
5. บันทึกรายการธุรกรรม
   ↓
6. สร้างงบประมาณ (Spending / Savings / Reward)
   ↓
7. ติดตามผ่านแดชบอร์ดและรายงาน
   ↓
8. กระทบยอดบัญชีเมื่อจำเป็น
   ↓
9. เพิ่ม LINE Bot เป็นเพื่อนเพื่อรับแจ้งเตือน
```

## คำถามที่พบบ่อย

**Q: สมาชิกที่เป็น Child ทำอะไรได้บ้าง?**
A: ดูข้อมูลบัญชี บันทึกรายการธุรกรรม และดูงบประมาณได้ แต่ไม่สามารถสร้างบัญชีหรือจัดการสมาชิกได้

**Q: ลืมบันทึกรายการทำยังไง?**
A: กระทบยอดบัญชีแล้วบันทึกยอดจริงได้เลย ไม่จำเป็นต้องย้อนกลับไปเติมรายการที่ขาด

**Q: อยากเปลี่ยนชื่อครอบครัวทำยังไง?**
A: ฟีเจอร์นี้อยู่ในระหว่างพัฒนา ให้ติดต่อผู้ดูแลระบบ

**Q: invite code หมดอายุแล้วทำยังไง?**
A: ให้ผู้ปกครองสร้าง invite code ใหม่จากหน้า "สมาชิกครอบครัว"
